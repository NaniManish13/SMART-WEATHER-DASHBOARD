# 🌦️ Smart Weather Dashboard

A desktop weather application built using Python and PyQt5 that displays real-time weather data.

## 🚀 Features
- 🌡️ Temperature (Celsius & Fahrenheit toggle)
- 💧 Humidity, Wind Speed, Pressure
- 📊 Graph visualization
- 🎨 Modern UI with emojis
- ⚡ Fast response with caching

## 🛠️ Tech Stack
- Python
- PyQt5
- Requests
- Matplotlib
- OpenWeather API

## 📦 Installation

```bash
git clone https://github.com/NaniManish13/weather-app.git
cd weather-app
pip install -r requirements.txt
python main.py
